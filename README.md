# AI-Powered Semantic Product Search System

Semantic search engine using Sentence Transformers (BERT) and cosine 
similarity — architecturally aligned with Retrieval-Augmented Generation 
(RAG) pipelines used in enterprise AI applications.

## Problem Statement
Traditional keyword search fails to understand user intent. This system 
enables context-aware retrieval that understands meaning, not just words.

## Tech Stack
- Python, Django
- Sentence Transformers (BERT)
- Scikit-learn (Cosine Similarity)
- NumPy, Pandas

## Key Features
- Context-aware semantic retrieval beyond keyword matching
- Typo tolerance and semantic matching
- Hybrid filtering and ranking mechanism
- Scalable Django REST backend

## Real-World Relevance
- Foundational to RAG (Retrieval-Augmented Generation) pipelines
- Applicable to banking policy search and document intelligence
- Enterprise knowledge base retrieval systems

## How It Works
1. Input query encoded using Sentence Transformers (BERT)
2. Cosine similarity computed against product embeddings
3. Hybrid filtering applied for improved relevance ranking
4. Results returned with semantic confidence scores
